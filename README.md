## üìñ About The Repository

This repository is a journey through the fundamentals of **Web Development**. It contains **15+ distinct projects** built entirely with **HTML, CSS, and Vanilla JavaScript**.

Unlike framework-based projects (React, Vue, etc.), these mini-apps focus on **first-principles JavaScript**:

- **DOM Manipulation** ‚Äì Creating, updating, and deleting elements dynamically  
- **Algorithmic Logic** ‚Äì Using `Math`, `Date`, and conditional reasoning  
- **State Management** ‚Äì Handling data without external libraries  
- **Event Handling** ‚Äì Mastering events, bubbling, delegation, and interactions  
- **Asynchronous Programming** ‚Äì Consuming external APIs using `Fetch` and `Async/Await`
